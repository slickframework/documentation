Features

Easy configuration

You use plain PHP arrays to set your application configuration. Extend it if you want to implement a different configuration format or service.

Form building

Simply create a Form object, define its field elements, validate and filter chains and render it in your view. All HTML will be created for you.

Simple ORM

A very simple and lightweight ORM component that uses PDO for database access. It implement the Active Record design pattern.

Data Filters/Validators

A very handy API to filter and validate data. This components define simple interfaces that you can use to implement your own filter and/or validator.

Flexible template engine

The default template engine uses Twig to render its view, a very powerful piece of software.

Open source

You are welcome to use, add or fork this PHP library. It has a very flexible licence even for commercial use.
